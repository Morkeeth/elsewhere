# Multi-witness ablation protocol

Elsewhere must prove that independent witnesses create useful disagreement, rather than simply adding calls.

## Fixed fixture

Use the supplied London decision and its parent-care shock. Keep the deterministic four-future payload, model, reasoning effort, output categories, and experiment constraints fixed.

## Compare

1. **Deterministic fallback:** existing fixed `trades-off` rows.
2. **One structured multi-lens call:** one model response creates every matrix row in one context.
3. **Independent witnesses:** the production architecture—four concurrent core calls and one synthesis.

Context Layers are excluded from the comparison. They are an optional user-authored extension, not part of the core four-lens claim.

## Measure

For each run, save the structured witness response and score it with `measureWitnessDisagreement`:

- complete option coverage;
- distinct assessment patterns;
- baseline/shocked cells where lenses disagree;
- non-fallback response rate.

Then conduct blinded human review against the same rubric:

1. Did it surface a decision assumption absent from the generic advice baseline?
2. Is the disagreement attributable to different protected values?
3. Is the selected uncertainty relevant and non-prescriptive?
4. Would the resulting fourteen-day experiment produce new evidence?

## Success threshold

Independent witnesses should outperform the one-call and deterministic conditions on both meaningful disagreement and human-rated experiment usefulness. If they do not, Elsewhere must narrow its claim to a visual interpretation layer rather than a necessary multi-agent instrument.

The evaluator intentionally does **not** equate more disagreement with better advice. It reports observable structure; people judge whether the conflict is useful.
